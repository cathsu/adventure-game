PROJECT TITLE: Adventure Game 

CLASS: CSIS 10C Programming Methods 1.5: C and C++ (Fall 2017)

PURPOSE OF PROJECT: Text-based adventure game that allows the user to create, connect, and move through different rooms. Have worked on the project intermittently since its due date. Extra functionality include:  
  a) saving and re-uploading the game. The game is saved to savedGame.txt
  b) creating items to put in rooms 
  c) carrying around items in a bag 
May or may not add extra features as time goes on.
